📖 Como Baixar e Jogar IsaacBlocks

  1. Baixe Python 3.13.7 (64 Bits)

    🐍 Baixar Python 3.13.7
  2. Depois que baixar o Python, vá em Download-Directory-Github

    📦 Abrir Download-Directory-Github
  3. Cole isto no Download-Directory-Github: https://github.com/franciscoeisaac890lindo-afk/IsaacBlocks-Download-Py/tree/main/IsaacBlocks
  4. Depois abra o arquivo .zip baixado e vá em: dist
  5. Abra: IsaacBlocks.exe
  6. 🎉 Pronto! Agora você está jogando IsaacBlocks.

  • Windows 10 ou Windows 11
  • Python 3.13.7 (64 Bits)
