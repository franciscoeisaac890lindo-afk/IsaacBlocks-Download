📖 Como Instalar

  1. Baixe o Python 3.13 (64 Bits)

    Baixar Python
  2. Vá para: https://download-directory.github.io/
  3. Cole este link: https://github.com/franciscoeisaac890lindo-afk/IsaacBlocks-Download-Py/tree/main/IsaacBlocks
  4. Faça o download do projeto.
  5. Extraia a pasta baixada.
  6. Abra o CMD na pasta do jogo.
  7. Digite: python isaacblocks.py
  8. Pronto! Você acessou o aplicativo e fez o download.
