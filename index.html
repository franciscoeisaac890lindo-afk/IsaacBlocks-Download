📖 Como Baixar e Jogar IsaacBlocks

  1. Baixe Python 3.13.7 (64 Bits)

    🐍 Baixar Python 3.13.7

  2. Depois que instalar o Python, vá em Download-Directory-Github

    📦 Abrir Download-Directory-Github

  3. Cole este link no Download-Directory-Github: https://github.com/franciscoeisaac890lindo-afk/IsaacBlocks-Download-Py/tree/main/IsaacBlocks

  4. Depois abra o arquivo .zip baixado e vá na pasta: dist

  5. Abra o arquivo: IsaacBlocks.exe

  6. 🎉 Pronto! O IsaacBlocks foi iniciado com sucesso.
